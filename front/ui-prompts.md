# Prompts UI — Pages principales

## PROMPT GLOBAL
Tu es expert UI/UX + DA + lead dev Next.js. Objectif : e-commerce premium style “editorial minimal luxe”.
Contraintes : accessible, responsive, performant, SEO, états d’erreur/loading.
Livrables : structure UI, microcopy FR, animations (Framer Motion, GSAP, Lenis), états, règles design.
Style : sobre, très premium.
