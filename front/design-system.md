# Design System — Editorial Minimal Luxe

## Couleurs
- Fond : #F6F6F4 (blanc cassé)
- Noir profond : #181818
- Gris doux : #E5E5E5, #BDBDBD
- Accent chrome/neon : #C0C0C0, #00FFE7

## Typographie
- Titres : serif éditoriale (ex : Playfair, DM Serif)
- Texte UI : sans moderne (Inter, Neue Haas)

## Layout
- Grille responsive, gros espaces blancs
- Hiérarchie typographique forte
- Cards nettes, arrondis soft

## Boutons
- Minimal, arrondis, hover glossy
- Couleurs inversées au hover

## Micro-interactions
- Underline animé, blur léger, parallax subtil
- Hover image : zoom + glow doux
- Hover bouton : micro-shine

## Animations
- Reveal titres : SplitText (GSAP) ou mask ligne par ligne
- Images : blur-up + fade + micro parallax
- Cards : hover lift (2–6px), shadow douce, highlight gloss
- Scroll : stagger 0.06–0.12s, easing power3.out
- Lenis : smooth scroll léger
- Transitions pages : fade + slide

## Accessibilité
- Contrastes respectés
- Focus visible
- Navigation clavier
- Aria labels sur composants interactifs

## Responsive
- Mobile-first, breakpoints : sm/md/lg/xl
- Grille adaptative, typographie fluide

---
Ce guide est à appliquer sur tous les composants/pages du projet.
